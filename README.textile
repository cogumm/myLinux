h1. Home

Script de instalação de todos os aplicativos e hambiente de desenvolvimento do meu Ubuntu 14.x

h2. Mantenedor

* Gabriel F. Vilar - "CoGUMm.net":http://cogumm.net


h2. Modo de usar

h3. To install it, just execute the following command:

<pre>git clone git@github.com:cogumm/myUbuntu.git</pre>

<pre>
cd myUbuntu
sh myUbuntu.sh
</pre>

<pre>
cp myUbuntu/home/.bashrc ~/
cp myUbuntu/home/.gitconfig ~/
</pre>
